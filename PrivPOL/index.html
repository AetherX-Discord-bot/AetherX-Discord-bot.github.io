AetherX Bot Privacy Policy

At AetherX Bot, we are committed to transparency regarding the data we collect and how it is used. Below is an overview of our data practices:

1. Data Collection

AetherX Bot collects the following information:

  • User ID + Server:
    • Level
    • XP
  • User ID:
    • Bio
    • Favorite Color
    • Affiliation
    • AllowStatusView: Allows a user to see the activities of the user via the $profile command
    • DefAllLongOrShort: Toggles the $profile command to show detailed or minimal information

All other data is retrieved from Discord's public information, and is used strictly for the Bot’s functionality.

2. Data Security

The data we collect is used solely to enhance the user experience and improve Bot functionality. This data is stored securely and only accessed as needed for Bot management.

3. No Data Sharing

Since AetherX Bot does not share or sell any user data to third parties, all collected data is kept within the Bot’s internal configuration and is used solely for Bot functionality.

4. No Data Retention

AetherX Bot does not retain user data beyond its immediate use for Bot-related purposes. All data is stored temporarily as necessary and is not kept longer than required.

5. Changes to this Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in functionality or compliance requirements. Any updates will be communicated via appropriate channels, such as the Bot or its support server.
