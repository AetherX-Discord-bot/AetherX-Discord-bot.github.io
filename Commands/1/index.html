General Commands

!changelog - Display changes made to the bot.
!request - Request a feature or features to be added.
!afk - Go afk and set a custom message.
!ping - Displays the bot latency and API latency.
!help - Shows this help menu with all available commands.
!yippee - Sends a "Yippee" GIF.
!global [message] - Sends a message to all servers where the bot is present.
!about or !info - Provides information about the bot, developer, and invites.
!profile / !profile @user / !profile (user ID) - See information on a user's profile.
